READ ME

Projekt symuluje i wyświetla ruch trzech ciał z ładunkami kulombowskimi. Jest opcja obracania i przybliżania kamery oraz wyłączenia jej automatycznego ruchu. W programie zawarte są masy, ładunki i początkowe wektory prędkości. Pozwala to na ich zmianę.

Wymagania:
-Python
-scipy
-numpy=matplotlib

Instrukcja obsługi:
-upewnij się że masz wymagane biblioteki
-zapisz skrypt jako 3BP.py
-odpal z terminala 

Początkowe kondycję:
-W kodzie są opisane masy, ładunki, pozycje i wektory ciał. Można je dowolnie zmieniać
